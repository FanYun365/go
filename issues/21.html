  • 宝宝生活记录 (Baby Tracker) (http://www.nighp.com/babytracker/)
    --（喂奶、换尿布、睡眠，婴儿成长笔记）( iOS / Andr​​oid / @APKPure应用市场 )

（这里强烈推荐一下，免费版就很好用，不过界面下方有个讨厌的广告，我自己买了无广告的Pro版本），这样心心的每次活动我们都会记录下来，很有意思的是这个APP的数据可以以CSV格式导出（这个太棒了！），而且它自身就可以生成各种的报告，报告还可以以PDF格式导出并发送给其他应用。

从女儿心心出生开始，我们就通过各种方式记录她的各种信息：睡眠记录，吃药记录，体温记录，换尿布记录，哺乳记录等等。毕竟，处于忙乱状态的人们是很难精确地回忆各种数字的，特别是在体检时面对医生的询问时。大部分父母无法准确回答小孩上周平均的睡眠时间，或者平均的小便次数，这在很多时候会影响医生的判断。 （《生活中的数据可视化之 -- 换尿布》 - I code it ）

内容提要 👍

“宝宝生活记录”专注于记录和统计新生宝宝的日常生活，帮助您定量了解和分析宝宝成长状况，并能随时记录宝宝生活中的趣事。

“宝宝生活记录”提供两个版本：
版本一：免费版
可永久免费使用绝大部分功能，包括数据同步功能；可在14日内免费使用全部功能；三周后将显示广告，并锁住“统计图表”功能。
版本二：内购升级为完整版
无广告，完整功能

1、功能全面
“宝宝生活记录”能对宝宝的生活轻松记录、编辑和统计分析，并能在安卓或iOS多设备间同步数据，也能方便的导出、打印数据和图表。“宝宝生活记录”包括宝宝今日（主页）、记录、编辑查看、统计四大主要模块，具体包括如下内容：

  • 哺乳喂食--可记录母乳，母乳瓶喂，配方奶，辅食的记录和统计，能自定义食物种类和数量；母乳可计时方式记录；能设置闹钟提醒下次喂食时间。
  • 换尿布--可记录嘘嘘、便便、混合和干爽四种状态，并能设置闹钟提醒下次换尿布时间。
  • 睡眠--可计时方式记录，并能设置闹钟提醒下次睡眠时间。
  • 成长--除了记录宝宝身高、体重、头围数据，更在统计图中反映世卫组织（WHO）百分位曲线，方便评估宝宝的成长状态。
  • 里程碑--可记录宝宝成长过程中的重要事件，通过拍照来留下美好瞬间。
  • 欢乐时光--通过照片和文字来记录宝宝生活的点点滴滴。
  • 其它活动--可自定义适合宝宝的任何记录项，可计时方式记录。
  • 体温记录--发热时可以通过体温记录来评估宝宝的身体状况。通过设定闹钟来提醒定时测量
  • 记录和提醒喂药--通过闹钟来提醒定时喂药，并在需要时能查询详细喂药记录。
  • 接种疫苗

2、方便易用
“宝宝生活记录”的丰富功能均能方便快速地实现。让您记录、查看、编辑等操作能通过最简单的步骤完成。

3、支持记录多个宝宝的活动
可以方便地记录和管理多个宝宝的生活

4、完善的数据同步和备份
通过我们的专用服务器可以轻松地在不同安卓设备间或iOS设备间同步和备份数据。

5、支持数据和图表的打印、email、导出（csv格式）
